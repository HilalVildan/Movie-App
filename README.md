# React Movie App

Responsive Movie App made using React and Tailwind.

<hr />

## What I Used

- Firebase
- React Router Dom
- React Toastify
- Tailwind

### Installation Options

```
$ git clone https://github.com/HilalVildan/Movie-App.git
```

```
$ npm install
```

```
$ npm start
```

### Or

```
$ git clone https://github.com/HilalVildan/Movie-App.git
```

```
$ yarn
```

```
$ yarn start
```
